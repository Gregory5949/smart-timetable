Набор данных для мероприятий весеннего семестра 2022 года.
Все файлы в формате csv с раздлителем ",".
Связи между объектами можно восстановить через поля id, например, shedule_id связано с schedule.

academic_calendar_irregular_rule - праздничные дни
building - учебные корпуса
building_distance - время, требуемое для перехода между корпусами
course_unit - курсовые сборки
cycle_realization - циклы
event - мероприятия
event_attendee - участники мероприятий
event_type - типы мероприятий
grid - сетки расписания
grid_slot - слоты сетки расписания
room - аудитории
room_type - типы аудиторий
schedule - расписания
